# go-game-client

__Building a go game to learn some clojure. Forgive its many shortcomings.__


## Run It

`cd /path/to/cloned/repo`

`lein cljsbuild once`

`lein ring server`

`open "http://localhost:3000"` (<- should happen automatically)
